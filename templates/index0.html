API Reference

Simple, RESTful API endpoints for virtual try-on integration

POST /upload/person

Upload a person image to use for try-on. Requires email and image file.

curl -X POST http://localhost:8080/upload/person \
  -H 'Authorization: Bearer client_id:client_secret' \
  -F 'email=customer@example.com' \
  -F 'person_image=@/path/to/person.jpg'

POST /upload/garment

Upload a garment image to try on. Requires product ID and image file.

curl -X POST http://localhost:8080/upload/garment \
  -H 'Authorization: Bearer client_id:client_secret' \
  -F 'product_id=tshirt-001' \
  -F 'garment_image=@/path/to/garment.jpg'

POST /run-tryon

Run the try-on process with previously uploaded images.

curl -X POST http://localhost:8080/run-tryon \
  -H 'Authorization: Bearer client_id:client_secret' \
  -H 'Content-Type: application/json' \
  -d '{
    "email": "customer@example.com",
    "product_id": "tshirt-001"
  }'

GET /preview-result

Get the try-on result image URLs.

curl -X GET 'http://localhost:8080/preview-result?email=customer@example.com&product_id=tshirt-001' \
  -H 'Authorization: Bearer client_id:client_secret'

Documentation

Simple steps to integrate with your website

Quick Start

Add the following code to your webpage to integrate the Virtual Try-On button and functionality:

<!-- 1. Add the Virtual Try-On Button wherever you want it to appear -->
<button id="virtual-tryon-button" class="your-button-class">Try On Virtually</button>

<!-- 2. Add the Virtual Try-On Script before closing </body> tag -->
<script>
  // Virtual Try-On configuration
  window.virtualTryOnConfig = {
    apiEndpoint: 'https://your-api-endpoint.com',
    clientId: 'your_client_id',
    clientSecret: 'your_client_secret',
    productId: '12345', // Your product ID
    productImageUrl: 'https://your-site.com/images/product.jpg', // Product image URL
  };
</script>
<script src="https://your-api-endpoint.com/static/js/virtual-tryon.js" async></script>
